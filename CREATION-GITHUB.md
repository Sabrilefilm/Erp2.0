### 1. Créer le repository sur GitHub
1. Allez sur : https://github.com/new
2. **Repository name** : `Erp2.0`
3. **Owner** : `UnionAgency` (votre organisation)
4. **Description** : `ERP 2.0 - Agence TikTok Management System`
5. **Visibility** : Private ou Public selon vos besoins
6. **NE PAS** cocher "Add a README file" (on l'a déjà)
7. **NE PAS** cocher "Add .gitignore" (on l'a déjà)
8. Cliquez sur **Create repository**
