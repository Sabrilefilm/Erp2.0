Logo pour la facture PDF
========================

Placez ici le fichier : logo-unions-agency.png

(Format PNG ou JPG, recommandé : PNG avec fond transparent, hauteur ~44px.)

Sans ce fichier, la facture affichera le texte "UNIONS AGENCY" à la place du logo.

Vous pouvez aussi définir un chemin personnalisé dans .env :
FACTURE_LOGO=/chemin/absolu/vers/votre-logo.png
